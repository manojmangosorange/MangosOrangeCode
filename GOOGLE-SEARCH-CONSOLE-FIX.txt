STEP 4: VERIFY .HTACCESS IS CORRECT
────────────────────────────────────────────────────────────────────────
Make sure your .htaccess in public_html/ has:

```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  
  # Ensure sitemap.xml and robots.txt are accessible
  RewriteRule ^sitemap\.xml$ - [L]
  RewriteRule ^robots\.txt$ - [L]
  
  # Don't rewrite files or directories
  RewriteCond %{REQUEST_FILENAME} -f [OR]
  RewriteCond %{REQUEST_FILENAME} -d
  RewriteRule ^ - [L]
  
  # Rewrite everything else to index.html for React Router
  RewriteRule ^ index.html [L]
</IfModule>
```

This is CRITICAL! Without this:
- sitemap.xml will redirect to your React app
- Google can't fetch it
- Shows "Couldn't fetch" error
